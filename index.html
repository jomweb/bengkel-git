Bengkel Git JOMWEB Telah hadir? Dah tersenarai?

Laman ini adalah tempat untuk peserta Bengkel Git JOMWEB membuktikan kefahaman terhadap apa yang dipelajari di dalam kelas. Bagi mereka yang telah menghadiri Bengkel Git oleh JOMWEB, anda dijemput untuk fork projek ini (https://github.com/jomweb/bengkel-git), tambahkan nama dan kemudian buat pull request

Sedikit panduan:

  • Fail senarai peserta dibuat mengikut tarikh dan pihak yang bekerjasama untuk menjayakan acara, seperti 20150808-nef.html untuk Bengkel Git JOMWEB pada 8 Ogos 2015, yang dijayakan bersama NEF.
  • Pastikan anda mengubahsuai fail yang betul.
  • Ubahsuai fail dan menambah nama anda (gunakan nama penuh) dan buat pautan ke akaun Github anda (contoh, https://github.com/kidino)
  • Pastikan pull request dibuat ke branch gh-pages
